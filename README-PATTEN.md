## UMENG 的各个依赖库

### For maven

maven库地址:

```
<repository>
    <id>github-srain-umeng-lib</id>
    <url>{repository_url}</url>
    <releases>
        <enabled>true</enabled>
    </releases>
    <snapshots>
        <enabled>true</enabled>
    </snapshots>
</repository>
```

现有的类库有:

```
{maven_dependencies}
```

---

### For gradle

依赖库地址:

```
maven {
    url '{repository_url}'
}
```

现有的类库有:

```
{gradle_dependencies}
```
