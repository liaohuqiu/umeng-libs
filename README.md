## UMENG 的各个依赖库

```
<repository>
    <id>github-srain-umeng-lib</id>
    <url>https://raw.githubusercontent.com/liaohuqiu/umeng-libs/master/repository</url>
    <releases>
        <enabled>true</enabled>
    </releases>
    <snapshots>
        <enabled>true</enabled>
    </snapshots>
</repository>
```
